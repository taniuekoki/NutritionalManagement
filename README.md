# NutritionalManagement
糖尿病や生活習慣病にならために簡単にわかりやすく栄養管理ができるWebアプリをSpringBootを用いて制作しました。
# 工夫した点
* わかりやすく操作しやすいUI
* 摂取量を円グラフで表示
* 健康によさそうな色を使用
* 気軽に使えるようにレスポンシブ対応
# 機能一覧
* ログイン・ログアウト機能
* 各栄養素とカロリーの摂取量が確認できる
* 食事登録・編集・削除・追加
* 食事検索
* 目標編集
# 実装環境
バックエンド：Java(SpringBoot),MySQL<br>
フロントエンド：HTML,CSS(BootStrap),Thymeleaf
